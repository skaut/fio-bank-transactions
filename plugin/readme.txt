== Description ==

<h2> Minimální požadavky</h2>
- WordPress 4.9.6 a vyšší
- PHP 7.1 a vyšší

Plugin pro zobrazování tabulky transakcí z Fio banky přes shortcode. Je možno přidávat více účtů a zobrazovat si je na různých stránkách a příspěvcích.

Zatím se jedná o BETA verzi, s verzí 1.0 se můžou některé věci změnit.

**Jsme na GitHubu**
[https://github.com/skaut/skaut-fio-bank-transactions/](https://github.com/skaut/skaut-fio-bank-transactions/)

== Installation ==
1. Stáhnout si plugin a aktivovat
2. Získat a zadat token podle [návodu.](http://napoveda.fapi.cz/article/144-jak-vygenerovat-token-ve-fio-bance)

== Frequently Asked Questions ==
**Jak plugin správně nastavit?**
Je potřeba vytvořit nový účet a zadat token podle [návodu.](http://napoveda.fapi.cz/article/144-jak-vygenerovat-token-ve-fio-bance)
